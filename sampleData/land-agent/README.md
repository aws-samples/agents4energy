# Land Agent Sample Data

## Overview
This directory contains sample data for the Land Agent, designed to demonstrate how AI agents can assist with oil and gas land management tasks. The data spans various aspects of land work including leasing, title examination, regulatory filings, visualization of ownership interests, and mapping.

## Directory Structure

- **leases/**: Contains sample oil and gas lease documents
- **title/**: Includes title opinions and chain of title documentation
- **maps/**: Contains GIS map data and visualizations
- **regulatory/**: Includes regulatory filing requirements and procedures
- **ownership/**: Contains ownership charts and interest calculations

## Use Cases

The Land Agent sample data enables landmen, land managers, and other users to:

1. **Lease Management**: 
   - Analyze lease terms, obligations, and expiration dates
   - Understand Pugh clauses, depth severance, and other specialized provisions
   - Example: "Explain the difference between Pugh clauses and depth severance provisions in a lease"

2. **Title Research**: 
   - Review title opinions and understand ownership chains
   - Identify curative requirements for title defects
   - Example: "What curative documents do we need for the Wilson lease title issues?"

3. **Spatial Analysis**:
   - Visualize lease positions on maps
   - Identify offset operators and nearby wells
   - Example: "Show me a map of our leasehold position in Sections 14-16, T4N-R8W, Lea County, NM"

4. **Ownership Calculations**:
   - Calculate net revenue interests and working interests
   - Visualize ownership distributions with charts
   - Example: "Calculate the net revenue interest for all working interest owners in the Johnson #2H well"

5. **Regulatory Compliance**:
   - Access permitting requirements by jurisdiction
   - Understand filing deadlines and procedures
   - Example: "What regulatory filings are required before spudding a well in Eddy County, New Mexico?"

## Visualization Capabilities

This data is designed to work with the Land Agent's visualization abilities, enabling:

1. **Interactive Maps**:
   - Property boundaries with Township/Range/Section grids
   - Color-coded lease ownership
   - Well locations and status
   - Infrastructure and facilities

2. **Ownership Charts**:
   - Pie charts for working interest distribution
   - Donut charts for revenue allocation
   - Bar charts for royalty breakdowns
   - Customizable visualizations for presentations

3. **Process Flowcharts**:
   - Permit application workflows
   - Title examination processes
   - Regulatory filing sequences

4. **Documentation Integration**:
   - Visual references linked to documentation
   - Highlighted language from lease provisions
   - Timeline visualizations for lease expirations

## Integration with Knowledge Base

This sample data is structured to be loaded into a knowledge base that the Land Agent can access. The agent uses this knowledge to:

1. Answer detailed questions about land management topics
2. Generate visualizations of ownership interests or lease positions
3. Explain regulatory requirements and processes
4. Assist with calculations for division orders or revenues
5. Interpret complex lease language and provisions

## Extending the Dataset

To add more data:

1. Follow similar formats to the existing files
2. Include realistic lease terms, ownership scenarios, and legal descriptions
3. Add sample plats, title opinions, and regulatory filings
4. Incorporate visualization specifications for additional properties

The goal is to demonstrate how natural language queries can democratize access to land data, enabling users to get insights through conversation rather than by navigating complex software or reading lengthy documents.
